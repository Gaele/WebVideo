
Pour faire marcher le site :
---Installer la base de donnee nommee "RequeteDeBase.sql"
---Activer le "rewrite_module" de apache (ou fonction equivalente sous autre serveur).

/!\ Attention les films ont ete supprimes pour pouvoir etre envoye par mail.
Pour faire fonctionner le site convenablement, il faut rajouter des videos dans le repertoire Web/video. Ces videos doivent avoir le meme nom que les images du sous-repertoire
Web/video/poster.

Le dossier GeneratePDF contient un code de generation des informations pdf client non incorpore au site.

Le module Application/Frontend/News contient dans ses vues (sous repertoire Vues) plusieurs fichiers indexes et shows.
Ceci vient du fait que les affichages faits exclusivement en bd ont ete gardes bien que le site passe a present par du xml.
Pour mieux se reperer, cf Application/Frontend/Config/routes.xml
